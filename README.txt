Contruct GAME - NW.js Packaged Version
------------------------------------

HOW TO RUN:
1. Download NW.js (Normal build) from https://nwjs.io/
2. Extract NW.js in a folder called "NWJS".
3. Copy ALL files from the NW.js folder (including nw.exe) into this ArcadeGame folder.
4. Export your Construct 3 project as "HTML5".
5. Put all exported files into the folder called 'game'.
6. Run run-game.bat.

WHAT THIS BUILD DOES:
- Boots directly to fullscreen.
- Locks in kiosk mode (players cannot exit).
- No window borders.
- Touch + keyboard + joystick input works.
- Perfect for arcade cabinets, kiosks, and machines.
